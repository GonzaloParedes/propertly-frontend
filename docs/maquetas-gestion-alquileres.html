Alquia — Sistema de diseño

Dirección visual para una app de gestión de alquileres pensada para propietarios e inmobiliarias pequeñas, con usuarios de más de 45 años. La estética es luminosa y cercana: color vivo con buen contraste, fondos tibios en lugar del gris frío, mucho aire, texto grande y redondeado, y estados que se leen sin depender del color. Usá el selector “Paleta” (arriba a la derecha) para ver toda la interfaz aplicada con cada una de las 3 paletas.

Paleta 1 — Fresca

Teal vivo sobre un fondo apenas menta. Es la paleta recomendada por defecto: alegre y confiable a la vez, con el mejor contraste general. Uso: el primario para botones principales, sidebar y enlaces; el coral secundario solo en detalles decorativos; el verde solo para “Pagado”; el ámbar para “Por vencer”; el rojo reservado a “Vencido” y alertas críticas.

Paleta 2 — Cálida

Coral terracota sobre un crema luminoso, con un teal fresco de acento. Transmite cercanía y energía; ideal si el producto apunta a propietarios individuales más que a inmobiliarias. Uso: el teal secundario aparece solo en detalles (íconos decorativos, gráficos), nunca en estados.

Paleta 3 — Vibrante

Índigo alegre, fondos lila muy claros y un rosa de acento. Es la variante más “fintech” y moderna: recomendada si se quiere cobrar un plan premium o apuntar a inmobiliarias. Uso: el índigo domina la navegación; el rosa secundario se usa para links, gráficos y elementos interactivos de segundo nivel.

Tipografía

Nunito Sans para todo el texto de interfaz: abierta, amable y muy legible en tamaños chicos, ideal para el público 45+. Quicksand (redondeada y suave) para títulos y cifras importantes: aporta calidez y un aire moderno sin perder claridad. Todos los números usan cifras tabulares para alinear montos en tablas.
