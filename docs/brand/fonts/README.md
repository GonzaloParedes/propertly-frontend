# Tipografías Alquia · v1.1

Ambas familias son libres (SIL Open Font License) y están en Google Fonts. No se incluyen los binarios; descargar siempre desde la fuente oficial para tener la última versión.

## Quicksand — títulos y cifras destacadas
- Descarga: https://fonts.google.com/specimen/Quicksand
- Pesos autorizados: Medium 500, SemiBold 600, Bold 700
- NUNCA se usa para tipear el logotipo (el logotipo es un dibujo, ver `../logos/`)

## Nunito Sans — interfaz y texto corrido
- Descarga: https://fonts.google.com/specimen/Nunito+Sans
- Pesos autorizados: Regular 400, SemiBold 600, Bold 700, ExtraBold 800

## Jerarquía (pantalla, base 17 px)
| Nivel | Familia | Peso | Tamaño | Interlineado |
|---|---|---|---|---|
| H1 | Quicksand | 700 | 28–32 px | 1.25 |
| H2 | Quicksand | 700 | 22 px | 1.3 |
| H3 | Quicksand | 700 | 19 px | 1.35 |
| Cifra destacada | Quicksand | 700 | 32–34 px | 1.1 (tabular) |
| Cuerpo | Nunito Sans | 400 | 17 px | 1.55 |
| Secundario | Nunito Sans | 400 | 15 px | 1.5 |
| Etiqueta/botón | Nunito Sans | 700 | 16–17 px | 1.2 |

## Fallbacks
- Quicksand → 'Trebuchet MS', 'Segoe UI', sans-serif
- Nunito Sans → 'Segoe UI', Arial, sans-serif
- Email (fuentes seguras): Verdana / Arial para cuerpo; Trebuchet MS para títulos

## Embed web
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Quicksand:wght@500;600;700&family=Nunito+Sans:wght@400;600;700;800&display=swap" rel="stylesheet">
```
