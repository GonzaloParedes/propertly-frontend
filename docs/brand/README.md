# Alquia · Identidad de marca v1.1

Estructura de entrega:

- `manual/` — PDF publicado y fuente HTML usada para editarlo y exportarlo
- `logos/svg/` — vectores para web y Figma (trazados puros, sin fuentes)
- `logos/pdf/` — vectores para imprenta y oficina
- `logos/eps/` — vectores editables en Illustrator
- `logos/png/` — rasterizados transparentes para usos que no admiten SVG
- `icons/` — favicon 16/32 + favicon.ico, Apple Touch 180, PWA 192/512
- `social/` — avatar 1024, app icon 1024, imagen Open Graph 1200×630
- `email/` — logos y plantillas compatibles con clientes de correo
- `fonts/` — familias, pesos, jerarquía y fallbacks (con enlaces de descarga oficiales)
- `color/` — paleta en Markdown, CSS custom properties y JSON

Los SVG, PDF y EPS incluyen todas las versiones aprobadas. Los PNG incluyen las variantes de uso rasterizado más frecuentes.

`color/alquia-colores.json` es la referencia canónica de color. El Markdown y los tokens CSS deben mantenerse sincronizados con ese archivo.

Ver CHANGELOG.md para el historial.
